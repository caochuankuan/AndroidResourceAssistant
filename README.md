# Android 资源助手 (Android Resource Assistant)

这是一个专为 Android 开发者设计的纯前端网页工具，旨在解决图标资源管理中的痛点：重命名繁琐、导入流程冗长、包体积优化不便。

## 🚀 核心功能

### 1. 本地项目一键管理

* **深度扫描**：只需选择 Android 项目根目录，工具将自动递归寻找 `src/main/res` 下的所有 `mipmap-*` 文件夹。
* **资源分组**：自动将不同分辨率（mdpi, hdpi, xxhdpi 等）的同名图标归为一组展示。
* **同步重命名**：点击图标名称即可编辑，修改后将自动同步重命名本地文件夹内所有分辨率的同名文件。
* **时间排序**：资源按最后修改时间逆序排列，最近操作过的图标始终排在最前。

### 2. ZIP 压缩包智能导入

* **快速识别**：支持直接读取从设计平台（如蓝湖、Figma）导出的 ZIP 包。
* **弹窗预览**：左图右文布局，支持点击缩略图查看高清大图，防止命名错误。
* **直接写入**：在弹窗中修改好名字后，点击“一键写入项目”，工具将解压并自动按分辨率覆盖/新建到本地项目目录。

### 3. 图片体积压缩 (瘦身)

* **双模式切换**：支持“按组压缩”（全尺寸覆盖）或“按图片压缩”（单文件精准优化）。
* **体积排序**：压缩列表按文件大小逆序排列，快速锁定占用空间最大的“元凶”。
* **可控级别**：提供从“高画质”到“极限压缩”的四个等级选择，通过前端 Canvas 技术直接回写优化后的数据。

### 4. 交互优化

* **实时搜索**：主网格与导入弹窗均支持实时关键字过滤。
* **防误触拦截**：锁定项目后，自动拦截浏览器刷新/关闭操作，防止丢失目录句柄。
* **Toast 提示**：所有操作均有非阻塞式的吐司提示（Toast），操作流程丝滑。
* **透明背景**：图片预览区域采用棋盘格背景，完美识别白色及透明图标。

---

## 🛠️ 技术实现

* **File System Access API**：用于在用户授权下直接读写本地项目文件。
* **JSZip**：在前端内存中完成 ZIP 包的解析、提取与重新打包。
* **Canvas API**：实现前端图片压缩与编码转换。
* **纯前端架构**：无需 Node.js 后端，无需安装环境，一个 HTML 文件即可运行。

## 📝 使用指南

1. **准备环境**：
* 请使用 **Chrome** 或 **Edge** 浏览器（需要支持 File System Access API）。
* 通过本地服务器运行（如 VS Code 的 `Live Server` 插件），直接双击文件打开可能受权限限制。


2. **锁定项目**：点击“选择项目目录”，授权网页访问你的 Android 项目根目录。
3. **管理资源**：
* 使用顶部**搜索框**定位图标。
* 点击图标下方的**蓝色文字**进行重命名。
* 点击右上角“**刷新**”同步外部（如 Android Studio）的文件改动。


4. **优化体积**：
* 点击“**⚡ 压缩**”，勾选占用空间大的资源。
* 选择压缩质量并执行，文件将直接在原路径下被替换。



---

## ⚠️ 注意事项

* **权限授予**：在执行写入操作时，浏览器会弹出安全询问，请务必点击“**保存更改**”。
* **命名规范**：工具内置了 Android 资源命名检查（仅限小写字母、数字及下划线）。
* **浏览器兼容性**：目前仅支持基于 Chromium 内核的桌面端浏览器。
